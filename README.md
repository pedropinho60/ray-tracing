# Introdução

Você deve colocar nesse repositório todo o código fonte necessário para compilar e executar seu _Ray Tracer_.

Não esqueça de adicionar nestes arquivo `README.md`:

1. Instruções sobre como compilar o projeto.
2. Identificação dos componentes da equipe.
3. Indicação do que foi implementado, em relação ao que foi solicitado nos projetos.
   Se houver alguma limitação ou incompletude das _features_ solicitadas, explique-as nesse tópico.
4. Indicação se alguns dos itens extras foram implementados. Informa também como ativar o item extra.

> [Note!]
> Lembro que toda equipe deve implementar pelo menos 1 item extra até o final do semestre.
