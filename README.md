# Introdução

Raytracer desenvolvido como projeto de avaliação para disciplina de Computação Gráfica I.

# Compilação

Para compilar o projeto, execute os comandos a seguir:
```
mkdir build
cmake -B build
cmake --build build
```

É necessário ter o g++ e CMake instalados.

# Execução

Para executar o projeto, execute o comando a seguir:
```
./build/rt3 <arquivo_de_cena>
```
